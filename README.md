## 📸 معرفی اپلیکیشن

این اپلیکیشن یک مجله آنلاین کامل است که با **React Native** و **Expo** ساخته شده. کاربران می‌توانند مقالات رو مطالعه کنن، دسته‌بندی‌ها رو ببینن، کامنت بذارن و تجربه کاربری لذت‌بخشی داشته باشن.

---

## ✨ امکانات کلیدی

| بخش | توضیحات |
|------|---------|
| 🔐 **احراز هویت** | ورود و ثبت‌نام کاربران با ذخیره امن توکن |
| 📖 **مقالات** | نمایش لیست مقالات با کارت‌های زیبا و جذاب |
| 📂 **دسته‌بندی‌ها** | مرتب‌سازی مقالات بر اساس موضوعات مختلف |
| 💬 **سیستم کامنت** | ثبت و نمایش نظرات کاربران (فقط کاربران لاگین) |
| 🏠 **صفحه اصلی** | بخش‌های مختلف با دیتای واقعی از سرور |
| 🎨 **دیزاین مدرن** | گرادیان، سایه، انیمیشن و تجربه کاربری عالی |
| 📱 **ریسپانسیو** | سازگار با تمام دستگاه‌ها و اندازه‌های صفحه |
| 🔄 **رفرش خودکار** | قابلیت کشیدن به پایین برای بروزرسانی |
| 🗂️ **نویگیشن پیشرفته** | دراور منو و استک نویگیشن |

## 🚀 نصب و اجرا

### پیش‌نیازها

- Node.js (نسخه 18 یا بالاتر)
- npm یا yarn
- Expo CLI
- اندروید استودیو یا Xcode (برای شبیه‌ساز)

### مراحل نصب

```bash
# 1. کلون کردن پروژه
git clone [https://github.com/your-username/magazine-app.git](https://github.com/danialmousavi/ArticleHub.git)
cd magazine-app

# 2. نصب وابستگی‌ها
npm install
# یا
yarn install

# 5. اجرای پروژه
npx expo start
```
## 📝 نکات فنی

### 🏗️ معماری و ساختار

| بخش | توضیحات |
|------|---------|
| **مدیریت state** | استفاده از Context API برای مدیریت وضعیت لاگین و اطلاعات کاربر |
| **نویگیشن** | ترکیب Drawer Navigator برای منوی اصلی و Stack Navigator برای صفحات عمیق |
| **سرویس‌ها** | جدا کردن منطق ارتباط با سرور در پوشه services |
| **کامپوننت‌ها** | استفاده از کامپوننت‌های کوچک و قابل استفاده مجدد |

### 🔐 امنیت

- **ذخیره توکن** در `expo-secure-store` (ذخیره‌سازی رمزنگاری شده)
- **ولیدیت توکن** با سرور در هر بار اجرای برنامه
- **ارسال توکن** در هدر Authorization برای درخواست‌های نیازمند احراز هویت
- **حذف خودکار توکن** در صورت نامعتبر بودن
